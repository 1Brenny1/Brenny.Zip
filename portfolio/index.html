Anti-Cheat

Basic Anti-Cheat which uses a salted hash to ensure security.

This Anti-Cheat can detect several client modifications such as: speed hacks, jump hacks, noclip hacks, health hacks, btool hacks, fly hacks and core gui injects

To prevent cheaters from bypassing the anti-cheat, the anti-cheat uses constant pings with a token to sync server values to client values. The token produced is a salted sha256 hash which will NEVER equal the same value again. If the server detects that the ping was not recieved the user is kicked, however if the ping is detected to be modified the user is permanently banned.
